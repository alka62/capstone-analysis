# capstone-analysis
# 📊 Customer Churn Analysis & Prediction

## 📌 Project Overview
This project focuses on analyzing customer churn data to identify patterns and predict whether a customer will leave the service. The goal is to help businesses reduce customer loss and improve retention strategies using data-driven insights.

---

## 🎯 Objectives
- Analyze customer behavior using data analysis techniques  
- Identify key factors responsible for customer churn  
- Build a machine learning model to predict churn  
- Provide actionable business recommendations  

---

## 📁 Dataset
- File: `customer_churn.csv`  
- Rows: 500+  
- Features include:
  - Gender  
  - Tenure  
  - Contract Type  
  - Monthly Charges  
  - Churn (Target Variable)  

---

## ⚙️ Tools & Technologies
- Python 🐍  
- Pandas  
- NumPy  
- Matplotlib  
- Seaborn  
- Scikit-learn  

---

## 🔍 Project Workflow

### 1. Data Cleaning
- Removed missing values  
- Converted categorical data into numerical format  

### 2. Exploratory Data Analysis (EDA)
- Churn distribution analysis  
- Gender vs churn comparison  
- Contract type analysis  
- Monthly charges impact  

### 3. Data Visualization
- Count plots  
- Box plots  
- Heatmap  

### 4. Machine Learning Model
- Logistic Regression  
- Train-test split  
- Model evaluation using accuracy score  

---

## 📊 Key Insights
- Customers with **month-to-month contracts** have higher churn rates  
- Higher **monthly charges** increase the probability of churn  
- Customers with **low tenure** are more likely to leave  

---

## 🤖 Model Performance
- Algorithm: Logistic Regression  
- Accuracy: ~70–85%  

---

## 💡 Business Recommendations
- Encourage long-term contracts  
- Offer discounts to high-risk customers  
- Improve customer engagement strategies
